Website

A website is a collection of related web pages that are accessed through the internet using a browser (like Chrome or Safari).

It can show information, pictures, videos, or interactive tools.
Websites can be used for many purposes—like learning, shopping, sharing portfolios, or entertainment.

A static website

displays fixed content and does not change unless the developer updates the code. It is usually used for simple sites like personal portfolios or resume pages.

a dynamic website

shows content that can change based on user interaction or data from a database. Examples include social media platforms or blogs where users can log in, post content, and interact.

an e-commerce website

is designed for buying and selling products online. These sites allow users to browse items, place orders, and make payments—examples include Amazon and Flipkart.
